# Ansible Bind Role
An ansible role for installing bind.

[![Build Status](https://travis-ci.org/resmo/ansible-role-bind.png?branch=master)](https://travis-ci.org/resmo/ansible-role-bind)

## Usage:

    ---
    - hosts: all
      remote_user: root
      roles:
      - resmo.bind
